## AI 工具对比与选择

市面上的 AI 编程工具很多，选哪个好？别急，看完这一课你就清楚了。

### 四大主流工具

| 工具 | 最擅长 | 适合人群 | 免费额度 | 付费价格 |
|------|--------|---------|----------|----------|
| **Claude** | 代码质量最高，对话最自然，解释最详细 | 零基础/初学者 | 有 | $20/月 |
| **ChatGPT** | 用户最多，插件丰富，生态最完善 | 零基础/初学者 | 有 | $20/月 |
| **Cursor** | 专为编程设计的 AI 编辑器，深度集成代码库 | 有一定基础的用户 | 有 | $20/月 |
| **GitHub Copilot** | 在你写代码时实时补全，无缝嵌入 VS Code | 开发者/进阶用户 | 有 | $10/月 |

### 选工具决策树

按你的情况，这样选：

```
你是零基础、第一次接触编程？
  → Claude 或 ChatGPT 网页版（推荐 Claude，代码质量更稳定）
  → 不需要安装任何软件，浏览器打开就能用
  → 先学完本课程的前 4 章，再做调整

你已经有点基础、会打开文件编辑代码？
  → Cursor（下载安装后用 AI 直接编辑项目文件）
  → 适合第 6 章做项目时使用

你已经是程序员、想提高效率？
  → GitHub Copilot + VS Code（在你写代码时实时补全）
  → 可以配合 Claude/ChatGPT 做方案设计
```

### Claude Code（进阶推荐）

当你学完本课程、想更进一步时，可以试试 **Claude Code**。它是一个命令行 AI 编程工具，直接在终端里帮你写代码、调试、部署。不过这对零基础来说是超纲内容，先把网页版用熟再说。

### 我对初学者的推荐

**就用 Claude 或 ChatGPT 的网页版开始。** 原因很简单：

1. **零门槛**：浏览器打开就能用，不需要安装配置任何东西
2. **对话体验最好**：就像跟一个耐心的老师聊天
3. **免费额度够用**：学完本课程完全不需要付费
4. **避免认知过载**：你在学编程，不是在学工具。先用最简单的工具，把注意力放在学描述需求和迭代上

等你能熟练地用自然语言让 AI 做出网页后，再考虑是否需要 Cursor 或 Copilot。

### 常见误区

> "贵的工具一定更好？"——不一定。对于 Vibe Coding 入门来说，免费额度的工具已经足够。关键在于你描述需求的能力，而不是工具本身。用过三种工具的人不一定比用熟一种工具的人产出更好。

> "必须用我朋友推荐的那个？"——每个工具的核心能力（理解自然语言、生成代码）大同小异。选一个开始，比纠结选哪个更重要。
